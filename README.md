# Corporate Web Platform & CRM Prototype
**Company:** Ascendant Global INC. (AGI Group) | **Role:** Software Developer Intern

This repository serves as a technical documentation and architectural showcase of the systems developed during my internship.

## 🌐 Key Project: Corporate Web Platform
*   **Wix to Stack Migration:** Designed the company's official web presence, migrating from a Wix website to a custom stack using **Ionic, Angular, and Tailwind CSS**.
*   **UI/UX Design and Layout:** Designed high-fidelity, responsive layouts, maintaining a premium "Enterprise-tier" brand identity for the AGI Group.
*   **Dynamic Features:** Developed a functional booking consultation page that sends user information directly to the CRM system.

## 📊 Key Project: Customer Relationship Management (CRM) System
*   **System Architecture:** Developed a modular CRM prototype to centralize business operations, including lead tracking, deal pipelines, and client management.
*   **Sales Pipeline Engine:** Developed an interactive **Kanban-style Sales Pipeline** with drag-and-drop functionality and real-time weighted revenue forecasting.
*   **Security:** Designed a robust **Role-Based Access Control (RBAC)** system and a Global Activity Ledger to track every data modification.
*   **Real-time Updates:** Integrated **Socket.io** to provide live updates of notifications for new leads and booking requests.
*   **Automated Document Generation:** Created an automated **PDF generation engine (Puppeteer)** to transform deal data into professional proposals.
*   **End-to-End Lead Management:** Built a "Lead Dossier" system for centralized client history and automated qualification status tracking.
*   **Backend:** Built a robust REST API using **NestJS and Prisma ORM**, architecting a PostgreSQL schema to manage complex entity relationships.
